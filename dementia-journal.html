REWIND: AI-Powered Memory Support for Dementia Care

Developing an intelligent journaling and assessment tool that helps dementia patients maintain cognitive function through personalized memory exercises

Role

Full-Stack Developer & UX Designer

Timeline

Fall 2024 - Spring 2025

Technologies

Python, Flask, Firebase, Google AI (Gemini), HTML/CSS

Team

Independent Project

The Challenge

Dementia affects over 55 million people worldwide, with numbers projected to triple by 2050. While memory loss is progressive, research shows that regular cognitive exercises and memory retrieval practice can help slow decline and improve quality of life for patients.

Current memory support tools face critical limitations:

  • Generic exercises that don't connect to patients' personal experiences and memories
  • Lack of personalization making it difficult for patients to engage meaningfully
  • No adaptive difficulty to match patients' changing cognitive abilities
  • Poor caregiver visibility into patient progress and memory retention

These gaps mean that patients often lose interest in cognitive exercises, and caregivers lack the data needed to provide informed support and track disease progression.

Our Solution

REWIND combines daily journaling with AI-generated memory quizzes to create a personalized cognitive exercise system. By using patients' own diary entries as the basis for recall questions, the tool makes memory practice more engaging and meaningful while providing caregivers with actionable insights.

Key Features

Personal Diary System

Simple, accessible interface for patients to record daily experiences and thoughts

AI-Generated Quizzes

Google Gemini AI creates personalized memory questions based on diary entries

Adaptive Difficulty Levels

Four difficulty settings targeting different time periods (1 week to 3 months)

Progress Tracking

Comprehensive analytics showing quiz performance and memory retention over time

Secure Cloud Storage

Firebase integration ensures patient data is safely stored and accessible

Caregiver Dashboard

Planned feature for clinicians to monitor patient progress and cognitive trends

Development Process

User-Centered Design Approach

The design prioritized simplicity and clarity, recognizing that dementia patients may experience confusion or difficulty with complex interfaces. Large text, high contrast colors, and straightforward navigation were essential design principles throughout development.

Technical Architecture

The application was built using a modern web stack optimized for rapid development and scalability:

  1. Flask backend providing lightweight, flexible server-side logic
  2. Firebase Authentication for secure user management
  3. Firestore database storing diary entries and quiz results
  4. Google Gemini AI integration generating contextual quiz questions
  5. Responsive frontend ensuring accessibility across devices

AI Integration Strategy

The core innovation was connecting personal diary entries to AI-generated memory assessment. The system retrieves entries within a specified timeframe, sends them to Google's Gemini API with carefully crafted prompts, and receives back personalized quiz questions that test recall of specific details and experiences.
